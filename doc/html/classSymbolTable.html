Detailed Description

SymbolTable. The symbol table is responsible for managing 4 map objects. Each symbol(Variable, Constant, Function and Script) is identified by a name.

Only one symbol type per name may exist. When setting a symbol of an existing name with different type the original symbol is deleted.
