Detailed Description

Expression Parser. The expression parser evaluates expressions in string form using an optionally supplied symbol table.

The symbol table defines variables, constants, functions and scripts.

Variables can be changed from expressions using a special syntax and these changes are stored in the passed symbol table. Functions are implemented using exprtk's function_compositor. Scripts are implemented as a custom exprtk function.
